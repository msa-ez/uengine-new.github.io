Our Products

Process GPT

Process GPT는 생성 AI(LLM)의 기능을 활용하여 비즈니스 프로세스를 자연어로 정의하고 실행 가능한 모델로 변환합니다. 이를 통해 전통적인 BPM 작업을 단순화하고 비용과 노력을 줄일 수 있으며, LangChain과 Chat-GPT 모델을 통합하여 실행과 정의를 원활하게 합니다. 고급 데이터베이스와 다중 에이전트 시스템을 활용하여 유연하고 효율적인 프로세스 자동화를 제공하며, 비즈니스의 접근성과 효율성을 높이는 것을 목표로 합니다.

FEATURES

“

대화로 프로세스 정의

ㆍ생성형 AI를 통한 프로세스 정의 생성
ㆍ생성과 함께 가시화
ㆍ생성 후에 관리자가 세부 수정
ㆍ세부 수정사항도 자연어로 기술할 수 있으며 BPMN 프로세스 모델러가 익숙한 사람은 직접 수정
“

대화로 UI 생성

ㆍ기존 UI 개발은 우수한 프론트엔드 기술자를 요구
ㆍAI를 기반하면 최신 프레임워크 기반의 멀티스크린에 최적화된 업무 화면을 즉시 생성할 수 있고 생성된 화면을 원하는 대로 수정할 수 있음
“

대화로 시스템 연동

ㆍ기존 정보시스템 간 연동은 다양한 기술적/정치적 이유로 시간과 노력이 많이 소요
ㆍAI를 기반하면 내부 시스템들의 Open API Spec 을 해석하여 프로세스 데이터 값들과 어떻게 연동하면 될지를 자동으로 생성
“

대화형 프로세스 실행

ㆍ업무 단계별 담당자에게 업무 지시 및 독려
ㆍ업무 체크포인트 준수 여부 확인 후 자동 반려
ㆍ프로세스 진행 모니터링
ㆍ이메일 발송 등 시스템 액티비티 수행
“

에이전트가 초안 생성

ㆍ멀티에이젼트를 통한 프로세스 자동화
“

AI 음성 인식

ㆍ영업/임원직 등 잦은 이동이 불가피할 경우 운전중에도 음성 인식 기반 대화로 업무 처리 가능

Utilized technology

ㆍLLM Framework: LangChain, LangObject (github.com/uengine-oss/langobject)
ㆍModel: Chat-GPT4 (for Process Execution), Chat-GPT3.5 (for definition Generation, Agent Execution)
ㆍRAG: LlamaIndex
ㆍMulti-agent: Crew AI + LangGraph
ㆍDatabase: Supabase (Postgres) + Neo4J Graph DB

+ uengine6 BPMS
+ MSA Easy (github.com/msa-ez/platform)
